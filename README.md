# My Unreal Project

## Engine Version
Unreal Engine 5.x (Epic Games Launcher)

## Setup
1. Install Unreal Engine via Epic Games Launcher
2. Clone this repo
3. Open the `.uproject`
4. Let Unreal build assets as needed

## Notes
- This project uses Git LFS for Unreal assets
- Do not commit Generated folders (Binaries, Intermediate, Saved)
